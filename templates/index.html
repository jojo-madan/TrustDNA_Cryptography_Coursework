Cryptographic Chain
1. 🏛️ CA Admin → Setup CA → Register staff (RSA-2048 + ECC P-256 + PKCS#12)
2. 🔬 Lab Tech → Sign genomic report (SHA-256 + RSA-PKCS1v15 + Nonce)
3. 👨‍⚕️ Doctor → Verify report → Sign prescription (ECDH + AES-256-GCM + RSA)
4. 💊 Pharmacist → 6-step verification → Dispense medication
🛡️
Security Guarantees
✅ Authentication — X.509 v3 + PKI challenge-response
✅ Integrity — SHA-256 hash sealed in RSA signature
✅ Confidentiality — AES-256-GCM AEAD encryption
✅ Non-repudiation — RSA digital signatures
✅ Replay protection — Nonce + timestamp per document
✅ Revocation — CRL checked on every operation
